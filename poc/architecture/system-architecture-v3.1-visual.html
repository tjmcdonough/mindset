Non-Negotiable Rules
RULE 1
No Cross-Module Writes
Each module writes only to its own collection. Changes propagate via events.
RULE 2
Zero Peer Reads
Core modules read Foundation cache only. Cross-module context comes via event accumulation.
RULE 3
Foundation Down, Events Up
Synchronous reads flow downward. Events decouple the upward flow. No circular call chains.
RULE 4
Domain-Scoped Payloads
Events carry the producer's domain data. Not IDs only. Not the whole world.
RULE 5
Projections Never Queried by Core
Front-end reads projections. Core modules never import them.
RULE 6
Module = Folder Boundary
Only index.ts exports are importable. Everything else is internal.
